# 🍕 Pizza Sales Dashboard

## 📌 Project Overview
This project analyzes pizza sales data using Power BI to provide business insights through interactive dashboards.

## 🎯 Objectives
- Analyze sales performance
- Identify top and bottom-selling pizzas
- Monitor KPIs
- Understand customer ordering patterns

## 📊 KPIs
- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Pizzas per Order

## 📈 Dashboard Features
- Daily & Monthly Sales Trend
- Sales by Pizza Category
- Sales by Pizza Size
- Top 5 Best Selling Pizzas
- Bottom 5 Selling Pizzas
- Interactive Filters & Slicers

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- SQL
- Microsoft Excel
